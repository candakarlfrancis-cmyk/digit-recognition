# Handwritten Digit Recognition System

## 📌 Description
This project uses Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN) to recognize handwritten digits (0–9).

## 🚀 Features
- Draw digits using mouse
- Upload image for prediction
- Uses trained CNN model

## 🛠️ Installation
1. Clone the repository:
   git clone https://github.com/yourusername/digit-recognition-system.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   py -3.10 app.py
